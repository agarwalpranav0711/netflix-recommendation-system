This project analyzes Netflix movies and TV shows dataset and builds a movie recommendation system using Python.

## Features
- Data cleaning and analysis
- Visualization using matplotlib
- Content-based recommendation system
- Similarity search using TF-IDF and cosine similarity

## Tech Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## How it works
Movie descriptions are converted into vectors using TF-IDF.  
Cosine similarity is used to recommend similar movies.
